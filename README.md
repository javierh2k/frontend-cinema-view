## <a name="setup"></a> Getting Started

### Install Yarn via Homebrew (macOs)

```sh
$ brew install yarn
```

### Install package dependencies

```sh
$ yarn
```

### Build application

```sh
$ yarn build
```

### Run the application

```sh
$ yarn start
```
